Industrial Project | Global InfoVentures Pvt. Ltd.

CCTV-Based Face Recognition Attendance System

Engineered and deployed a real-time multi-stream face recognition system integrated with the G5 Portal. The system processes feeds from 20+ CCTV cameras using an optimized TensorRT-based SCRFD model for face detection and w600k_r50 for recognition, powered by NVIDIA DGX A100. Designed for educational institutions, the system ensures >99.99% accuracy, auto-identifies individuals, and updates centralized attendance logs reducing administrative load significantly.
